# Cursor 5分钟快速开始（0基础版）

**目标**: 5分钟内配置好Cursor，立即开始使用

---

## ⚡ 第1步: 设置自动执行（最重要）

**目的**: 不再频繁点击RUN

1. 按 `Ctrl + ,`（打开设置）
2. 搜索框输入: `auto-run`
3. 找到 **Auto-Run Mode**
4. 点击下拉菜单，选择:
   ```
   Run Everything (Unsandboxed)
   ```
5. 设置会自动保存
6. **关闭并重启Cursor**

✅ 完成后，AI可以自动执行所有命令，无需每次点击RUN确认

**三个选项说明**:
- ❌ `Ask Every Time` - 每次都要点RUN（默认，很麻烦）
- ⚠️ `Auto-Run in Sandbox` - 自动运行但有限制（部分功能需确认）
- ✅ `Run Everything (Unsandboxed)` - 完全自动（推荐，最方便）

---

## ⚡ 第2步: 记住3个快捷键

| 按键 | 功能 | 何时使用 |
|------|------|---------|
| `Ctrl + L` | 打开Chat（问答） | 想问问题时 |
| `Ctrl + I` | 编辑当前文件 | 修改单个文件时 |
| `Ctrl + Shift + L` | 打开Agent（自动执行）| 复杂任务时 |

✅ 记住这3个，足够日常使用

---

## ⚡ 第3步: 第一个任务（测试）

**验证Cursor是否配置正确**:

1. 按 `Ctrl + Shift + L`（打开Agent）
2. 输入:
   ```
   创建一个测试文件 test.txt，内容是"Cursor配置成功"
   ```
3. 按 `Enter`
4. 看到左侧文件列表出现 `test.txt` → ✅ 配置成功

## 🎯 日常使用模板

### 修复错误
```
修复以下错误：[描述问题]

要求：
1. 找到根本原因
2. 提供修复方案
3. 验证修复后正常
```

### 添加功能
```
添加功能：[功能描述]

要求：
1. 位置：[在哪添加]
2. 功能：[做什么]
3. 不影响现有功能
```

### 问问题
```
[问题]

要求：用简单的话解释，我是0代码基础
```

## 📌 记住这句话

**"精确描述需求 + 说明具体要求 = AI完美执行"**

- ❌ 差：帮我改改这个页面
- ✅ 好：在P4LAB页面右上角添加蓝色的"导出"按钮，点击后下载JSON文件

## 🆘 遇到问题？

1. **AI理解错了**: 按 `Ctrl + Z` 撤销，重新描述
2. **不知道怎么描述**: 按 `Ctrl + L` 问AI："我想做XX，应该怎么描述？"
3. **代码出错了**: 按 `Ctrl + Shift + L` 输入："修复所有错误"
