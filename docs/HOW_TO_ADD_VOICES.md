# 🎙️ 如何扩充音色库

## 📌 当前状态

- **现有音色数量**: 5个真实音色
- **需要扩充**: 按分类添加更多音色（动漫、方言、搞笑、名人、女声特色等）

## ⚠️ 重要说明

**无法自动批量获取音色ID**，原因：
1. Fish Audio音色库数据是通过JavaScript动态加载的
2. API需要认证token才能调用
3. 音色的"热度排名"需要人工判断

**因此必须手动从官网获取音色ID。**

---

## 🚀 获取音色ID的步骤

### 1. 访问Fish Audio音色库

打开浏览器访问：https://fish.audio/zh-CN/app/discovery

### 2. 浏览和筛选音色

- 使用搜索框搜索关键词（如"动漫"、"粤语"、"御姐"等）
- 使用分类筛选器
- 查看热度排行榜

### 3. 复制音色ID

找到喜欢的音色后：

1. 点击音色卡片进入详情页
2. 从浏览器地址栏复制音色ID

**URL格式示例**:
```
https://fish.audio/m/f695c96089384b9989f6520e77332065/
                   ↑
              这就是音色ID
```

**音色ID**: `f695c96089384b9989f6520e77332065`

### 4. 记录音色信息

对每个音色记录以下信息：
- **ID**: 从URL复制的长字符串
- **名称**: 音色名称（如"木齐"）
- **性别**: male / female / neutral
- **分类**: 推荐/男声/女声/名人/动漫/方言/搞笑/特色
- **标签**: 简短的风格标签（如"妩媚"、"撒娇"）
- **描述**: 适用场景描述

## 📝 添加到配置文件

打开 `src/configs/festival/voicePresets.ts`，在对应分类中添加新音色。

### 示例：添加动漫音色

在 `VOICE_CATEGORIES` 数组中添加新分类或在现有分类中添加音色：

```typescript
{
  id: 'anime',
  name: '动漫',
  icon: '🎭',
  order: 4,
  voices: [
    {
      id: 'abc123def456...xyz',  // 从URL复制的ID
      name: '纳西妲',
      gender: 'female',
      tag: '萝莉',
      avatar: '/images/avatars/nashida.jpg',  // 可选
      description: '原神角色，可爱萝莉音'
    },
    {
      id: 'def456ghi789...xyz',
      name: '钟离',
      gender: 'male',
      tag: '低沉',
      avatar: '/images/avatars/zhongli.jpg',  // 可选
      description: '原神角色，磁性男声'
    }
    // ... 继续添加更多音色
  ]
}
```

### 示例：添加女声特色音色

```typescript
{
  id: 'female-special',
  name: '女声特色',
  icon: '💃',
```
