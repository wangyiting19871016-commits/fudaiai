## 🔄 交互流程

### 标准流程图

```
首页（展示4个分类）
  ↓ 点击分类
分类页（展示该分类下的所有功能）
  ↓ 点击功能
功能页（根据功能类型显示不同页面）
  ├─ 图片类：模板选择页 → 上传照片页 → 生成中
  ├─ 文字类：文字输入页 → 生成中
  └─ 语音类：文字输入 + 声音选择页 → 生成中
  ↓ 生成完成
结果页（展示结果 + 操作按钮）
  ├─ 保存图片
  ├─ 分享
  ├─ 重新生成
  ├─ 添加文案（部分功能）
  └─ 转为语音（部分功能）
```

### 关键交互点

1. **首页导航**
   - 4个分类卡片点击 → 进入对应分类页
   - 礼包Banner点击 → 进入VIP购买页
   - 底部导航：我的作品、使用帮助、联系客服

2. **返回逻辑**
   - 分类页：返回按钮 → 首页
   - 功能页：返回按钮 → 分类页
   - 结果页：返回按钮 → 功能页（重新生成）

3. **错误处理**
   - 生成失败：显示错误信息 + 重试按钮
   - 网络错误：提示检查网络 + 重试按钮
   - 超出限额：提示购买VIP

## ⚠️ 限制与约束

### 技术限制

1. **生成速度**
   - 图片生成：2-5秒（取决于复杂度）
   - 文字生成：1-3秒
   - 语音生成：2-4秒
   - 运势卡片：<100ms（模板方式）

2. **文件限制**
   - 上传图片：最大10MB
   - 支持格式：JPG、PNG、WEBP
   - 推荐尺寸：1024x1024或以上

3. **并发限制**
   - 同一用户同时只能有1个生成任务
   - 排队机制（待实现）

### 业务限制

1. **免费次数**
   - 见各功能列表
   - 每日0点重置
   - 超出需购买VIP

2. **VIP权益**
   - 所有功能无限使用
   - 无水印
   - 优先生成
   - 价格：¥19.9/月
