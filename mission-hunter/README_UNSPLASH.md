# Unsplash 资产猎手使用指南

## 概述

Unsplash 资产猎手是一个替换原 Pinterest 模块的新组件，用于从 Unsplash 获取高品质视觉素材，无需登录即可获取高清原图。

## 核心功能

1. **无需登录**：利用 Unsplash 的公开下载链接规律，直接获取高清原图
2. **API 驱动**：使用 Unsplash API 进行精准搜索
3. **高品质素材**：获取 Unsplash 上的专业大片
4. **无缝集成**：与原有报告系统兼容，保持"左右双轴"对比结构

## 安装与配置

### 1. 获取 Unsplash API 访问密钥

要使用 Unsplash 资产猎手，您需要先获取 Unsplash API 访问密钥：

1. 访问 [Unsplash Developers](https://unsplash.com/developers)
2. 注册一个开发者账号
3. 创建一个新的应用
4. 获取您的 `Access Key`

### 2. 配置 API 密钥

在 `unsplash_scraper.py` 文件中，找到以下代码行并替换为您的 API 密钥：

```python
self.api_key = "YOUR_UNSPLASH_ACCESS_KEY"  # 需要替换为实际的 API key
```

## 使用方法

### 运行模式

Unsplash 资产猎手支持以下运行模式：

1. **u-link 模式**：分析 + Unsplash 抓取 + 汇总报告
   ```bash
   python run.py --mode u-link --keyword "电影感调色"
   ```

2. **crawl 模式**：抓取 + 分析（不包含 Unsplash 抓取）
   ```bash
   python run.py --mode crawl --keyword "电影感调色"
   ```

3. **analyze 模式**：仅分析（不包含抓取）
   ```bash
   python run.py --mode analyze --keyword "电影感调色"
   ```

### 批量运行

您可以使用 `--keywords` 参数进行批量搜索：

```bash
python run.py --mode u-link --keywords "电影感调色,胶片感滤镜,复古相机参数"
```

## 工作流程

1. **分析阶段**：根据输入关键词，分析小红书数据并生成流派关键词
2. **搜索阶段**：使用生成的英文关键词在 Unsplash 进行搜索
3. **下载阶段**：利用 Unsplash 公开下载链接规律直接获取高清原图
4. **报告生成**：生成包含 Unsplash 专业大片的 HTML 报告，保持"左右双轴"对比结构

## 报告结构

生成的 HTML 报告包含以下部分：

1. **原始素材墙**：展示小红书高赞帖子
2. **任务候选名单**：生成的调色方案任务
3. **技术研报 - 商业决策分析**：
   - 本周流量密码总结
   - 核心流派拆解（左右双轴对比）
     - 左侧：小红书爆款封面
     - 中间：AI 拟合的 HSL 调色参数
     - 右侧：Unsplash 专业大片
   - 高频技术痛点
4. **原始素材库**：所有有效素材的汇总

## 技术细节

### 高清原图获取原理

Unsplash 提供了公开的下载链接规律：`https://unsplash.com/photos/[ID]/download`

通过该链接，我们可以直接获取图片的高清原图，无需登录或其他认证。

### 搜索策略

1. 使用分析阶段生成的英文关键词
2. 优先获取横版图片，适合展示
3. 按相关性排序
4. 每个关键词获取 3 张高质量图片

## 注意事项

1. **API 调用限制**：Unsplash API 有调用频率限制，请合理使用
2. **图片版权**：Unsplash 图片遵循 Creative Commons Zero (CC0) 协议，可免费用于商业用途
3. **网络连接**：确保网络连接稳定，特别是在批量下载图片时
4. **存储空间**：高清原图占用较大存储空间，请确保有足够的磁盘空间

## 与原 Pinterest 模块的区别

| 特性 | Unsplash 模块 | Pinterest 模块 |
|------|---------------|----------------|
| 登录需求 | 无需登录 | 需要登录，限制较多 |
| 图片质量 | 专业大片，高清原图 | 质量参差不齐 |
| 抓取速度 | 快速，API 驱动 | 较慢，浏览器模拟 |
| 稳定性 | 高，API 可靠 | 低，易受反爬限制 |
| 图片版权 | CC0 协议，免费商用 | 版权复杂，需谨慎使用 |

## 故障排除

### API 调用失败

如果遇到 API 调用失败，请检查：

1. API 密钥是否正确
2. 网络连接是否正常
3. API 调用频率是否超过限制

### 图片下载失败

如果遇到图片下载失败，请检查：

1. 图片 ID 是否有效
2. 网络连接是否稳定
3. 磁盘空间是否充足

### 报告生成失败

如果遇到报告生成失败，请检查：

1. 分析阶段是否成功生成了搜索任务
2. Unsplash 抓取是否成功生成了结果文件
3. 模板文件是否存在且完整
