# 音色预览音频目录

## 说明

此目录存放音色预览音频文件。

## 文件命名规则

- `yangshi.mp3` - 央视配音
- `nvdaxuesheng.mp3` - 女大学生
- `leijun.mp3` - 雷军
- `wangkun.mp3` - 王琨
- `dingzhen.mp3` - 丁真

## 生成方法

### 方法1：使用脚本生成
```bash
npx ts-node scripts/generateVoicePreviews.ts
```

### 方法2：手动生成
1. 在语音生成页面选择音色
2. 输入祝福语：「恭喜发财，马年大吉！祝您新的一年身体健康，万事如意！」
3. 生成并下载音频
4. 重命名为对应的文件名
5. 放到本目录

## 临时方案

如果没有预览音频，卡片将显示"暂无预览"提示。
