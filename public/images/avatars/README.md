# 音色头像图片目录

## 说明

此目录存放音色的头像图片。

## 如何获取图片

### 方法1：从Fish Audio官网下载

1. 访问 Fish Audio官网：https://fish.audio
2. 搜索对应的音色（如：雷军、王琨、丁真等）
3. 右键保存音色的头像图片
4. 重命名为对应的文件名（如：`leijun.jpg`）
5. 放到本目录

### 方法2：使用Fish Audio API

音色头像URL格式：
```
https://fs.fish.audio/model/{voice_id}/avatar.png
```

示例：
- 央视配音：`https://fs.fish.audio/model/59cb5986671546eaa6ca8ae6f29f6d22/avatar.png`
- 雷军：`https://fs.fish.audio/model/aebaa2305aa2452fbdc8f41eec852a79/avatar.png`

## 文件命名规则

- `yangshi.jpg` - 央视配音
- `nvdaxuesheng.jpg` - 女大学生
- `leijun.jpg` - 雷军
- `wangkun.jpg` - 王琨
- `dingzhen.jpg` - 丁真

## 图片规格

- 格式：JPG 或 PNG
- 尺寸：建议 200x200 以上
- 文件大小：建议 < 100KB

## 临时方案

如果没有头像图片，将显示粉红色渐变背景。
